## The Unforgettable — "Ignition" (the signature)
Scroll IS the car's startup sequence. Hero = **poster-over-video** (poster is the LCP — fixes the animated-hero mobile-LCP ceiling): a static GT poster paints instantly; an abstract energy video loop plays behind for motion. On scroll-into-view, GSAP pins the Ignition stage and runs, **all driven by `ScrollTrigger.progress` (NEVER off video events/seek — cross-browser desync risk):**
- a **plasma charge-line** (SVG `stroke-dashoffset`) traces the GT silhouette,
- the headline **clip-path masks** in,
- a **HUD counter** ticks `0 → 412 km/h` (used ONCE here, then banned site-wide),
- the render parallax-scales as the loop color-grades from black to plasma.
True `video.currentTime` scrub is reserved for ONE short (2–5s) self-controlled window via a rAF loop that lerps a target toward progress, with a ~2s seekability timeout → freeze on poster + pure-GSAP overlays. Mobile: poster + autoplay loop, **NO scrub**.
**One moment of agency:** the Configure page — a 2D color/trim **skin-swap** (swap render + palette tokens + spec values) + ONE momentum **hover-rotate** detail over a pre-rendered set, with a plasma charge-transition between trims. (No live 3D / WebGL.)

## Aesthetic — cinematic, anti-rut
Council's sharpest catch: **void-black + single-neon + HUD is the NEW EV cliché.** Divergence fixes baked in:
- **Studio-light is a GENUINE light-mode act**, not a toggle — the Design act inside "The GT" and the Studio skin are real light surfaces (bone/ink), reached via a scroll color-transition, so the template visibly escapes the void+neon fingerprint.
- HUD/counter used **once** (Ignition), banned elsewhere.
- **Film grain** (generative SVG) on every dark surface so flat void never reads "generated." NO grids of any kind.
- Charge-line + every stat tie to **real spec data** (range, 0–100, kWh, torque), never decorative.
- Real designed copy everywhere, no lorem.

## Color — "Voltage" + 3 skins
**Midnight (default):**
```
--void:    #08080B   /* 60% canvas */
--surface: #101015   /* panels */
--raise:   #17171F   /* elevated */
--bone:    #ECEAE3   /* text */
--muted:   #8A8A92   /* secondary text (AA on void) */
--line:    #24242C   /* hairlines */
--plasma:  #F0257E   /* signature charge — glows/strokes/large/CTA only */
--plasma-2:#FF4D94   /* lighter plasma for AA text-on-dark where needed */
--plasma-rgb: 240,37,126
```
**Functional tokens (forms/states — NOT brand):** `--ok:#36D399` `--err:#FF5466` `--focus:#7FB2FF` (each verified ≥3:1 / AA on all skins).
**Carbon skin:** graphite `--void:#15151A --surface:#1D1D24`, same plasma.
**Studio skin (GENUINE light):** `--void:#EDEBE4 --surface:#F6F4EE --bone:#14141A --muted:#5C5C64 --line:#D8D5CC --plasma:#D11468` (darker plasma for AA on light).
Magenta is NEVER used as small body/link text on void or Studio (fails AA) — only large display, strokes, glows, CTA fills (with dark text).
Banned: grids, indigo/cyan as the secondary (EV-generic), purple gradients.

## 8 section archetypes (each a DIFFERENT scale + rhythm + medium + anchor; screenshot-diff before "done")
1. **Ignition hero** — full-bleed poster-over-video + GSAP.progress choreography (anchor: charge-line + HUD).
2. **Kinetic-type act** — oversized Hubot VW headline carrying a real spec line, morphs on scroll; zero imagery (anchor: the type).
3. **Pinned spec-choreography** — pin the GT render; numbers (range/0-100/kWh) scrub up with it as captions reveal (anchor: the pinned render + counting specs).
4. **Horizontal pinned design rail** — sideways gallery of design/detail renders with cursor-displacement on hover (anchor: must move horizontally).
5. **Sticky split-stat** — pinned media one side, stepped performance numbers advancing the other (NOT a card grid).
6. **Cinematic interstitial** — full-bleed plasma **color-flood** scene-change with a single line (anchor: the flood; transition IS the section).
7. **Editorial overlap collage** — asymmetric layered interior/exterior cut-outs with parallax depth (Studio-light act).
8. **Configurator CTA** — color-swap teaser → Configure/Reserve.
